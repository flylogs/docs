# Company Settings

The Company Settings page is the control center of your Flylogs environment. Every option on it applies **company-wide**: it changes what your pilots, instructors and staff can see and do across schedules, flights, logbooks, duty records and billing.

You will find it in the manager menu under **Company → Settings**. Only company managers can open this page.

{% hint style="info" %}
**Changes save automatically.** There is no Save button — every change is stored a moment after you make it, and a small confirmation appears at the bottom-right corner of the screen. Settings take effect for every user on their next page load; nobody needs to log out and back in.
{% endhint %}

### The settings tabs

The page is organized in eight tabs. Each one is described in detail on its own page:

| Tab | What it controls |
| --- | --- |
| [General](general.md) | Date, time, duration and unit formats, start of the week, and the emergency login switch |
| [Flights](flights.md) | Logbook locking, file retention, flight form options, pre-dispatch gates and flight permissions per user type |
| [Schedule](schedule.md) | Who can see and manage the schedule, calendar colors, operating hours, self-scheduling limits and cancellation windows |
| [Pilots](pilots.md) | Pilot currency requirements and documentation compliance rules |
| [Duty Limits](duty-limits.md) | Duty time and flight time limitations, rest requirements and duty record policies |
| [Billing](billing.md) | The internal billing system: defaults, VAT, billing rates and Stripe online payments |
| [Integrations](integrations.md) | Airbly and Fleet Manager (formerly Spidertracks) automatic flight imports |
| [API](api.md) | Flylogs API activation, API keys and the IP address whitelist |

### Plan availability

Some tabs are only available on certain subscription plans. When a tab is not included in your plan, its settings are hidden and a notice invites you to upgrade.

| Tab | Free | Club | Premium | Unlimited |
| --- | --- | --- | --- | --- |
| General, Flights, Pilots | ✓ | ✓ | ✓ | ✓ |
| Schedule | — | — | ✓ | ✓ |
| Duty Limits | — | — | ✓ | ✓ |
| Billing | — | ✓ | ✓ | ✓ |
| Integrations | — | — | ✓ | ✓ |
| API | — | paid activation | paid activation | included |

See [Account types](../account-types.md) for a full plan comparison.
